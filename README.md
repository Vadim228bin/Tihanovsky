### Задание 3

Исходное выражение: **(X ∧ (Y ∨ Z)) ∨ ((X → Y̿) ∨ Z)**

**Шаг 1: Упрощение формулы**
Обрати внимание, что во второй скобке над **Y** стоят две черты (**Y̿**). В алгебре логики двойное отрицание отменяет само себя (не "не Истина" = Истина). 
Поэтому **Y̿ = Y**. 
Для удобства перепишем формулу без двойного отрицания:
**`F = (X ∧ (Y ∨ Z)) ∨ ((X → Y) ∨ Z)`**

**Шаг 2: Порядок действий для таблицы**
У нас 3 переменные (X, Y, Z), значит в таблице будет 2³ = 8 строк. 
Разобьем формулу на промежуточные действия (столбцы):
1. **`Y ∨ Z`** (логическое ИЛИ)
2. **`X ∧ (Y ∨ Z)`** (логическое И — левая часть главного выражения)
3. **`X → Y`** (импликация / следование)
4. **`(X → Y) ∨ Z`** (логическое ИЛИ — правая часть главного выражения)
5. **Итог: Столбец 2 ∨ Столбец 4** (результат всей функции)

*Напоминание по знакам:*
* **`∨`** (ИЛИ) = 0, только если оба нуля.
* **`∧`** (И) = 1, только если обе единицы.
* **`→`** (Следование) = 0, только в случае `1 → 0`.
